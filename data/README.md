# Data

`cost_of_living_sri_lanka.csv` holds monthly prices in LKR for seven essential categories (Jan 2023 – Dec 2024).

The values are **synthetic but realistic** — built for learning, chart practice, and storytelling about affordability pressure. They are **not** official Consumer Price Index (CPI) figures.

For official inflation statistics, see the [Department of Census and Statistics, Sri Lanka](https://www.statistics.gov.lk/).
